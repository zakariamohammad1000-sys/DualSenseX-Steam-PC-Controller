## Overview
DualSenseX Steam PC Controller is a powerful tool that brings the full potential of the PlayStation 5 DualSense controller to PC and Steam games. It supports advanced adaptive triggers, haptic feedback, LED customization, gyro controls, Xbox emulation via ViGEmBus, and custom profiles. The application works as a modern alternative to DS4Windows, delivering low-latency input, full feature support, and seamless integration with Steam on Windows 11.

## Installation
### For Regular Users
1. Download and extract the archive
2. Run `DualSenseX.exe` as Administrator
3. Follow the setup guide
4. Connect your DualSense controller via USB or Bluetooth
5. Configure adaptive triggers, haptic feedback, and profiles
6. Enable Steam Input or ViGEmBus emulation and start gaming

## Troubleshooting
| Problem | Solution |
|--------------------------------|-------------------------------------------------------|
| App doesn't start | Run as Administrator and disable antivirus temporarily |
| Controller not detected | Reconnect via USB/Bluetooth and restart tool |
| Adaptive triggers not working | Check game support and update firmware |
| Haptic feedback missing | Enable haptic settings and restart Steam |
| Gyro / LED issues | Calibrate controller in settings |
| Input lag / emulation problems | Update ViGEmBus and run as Administrator |
| Windows 11 compatibility error | Update Windows and graphics drivers |
| Download / update failed | Use the latest release and check your internet connection |
